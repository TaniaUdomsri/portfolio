Amazon Seller Command Center (ASCC)

Data visualization, and real-time dashboard

Amazon Seller Command Center (ASCC) is a real-time dashboard for Seller feedback through How is my driving? tool on Amazon Seller Central. Amazon Seller Central is a web interface used by merchants or third-party sellers to sell their products directly to Amazonâ€™s customers. The dashboard is an internal tool that makes seller feedback readily accessible to stakeholders.

My responsibilities

Project lead, and Front-end engineer

Challenge

  • Only available for advanced users Before ASCC, to get Seller feedback on any of Seller Central pages, stakeholders need to know SQL to use the data warehouse or separately create their workbooks on Tableau. Those methods require knowledge, effort, and training. I discovered these problems by experiencing employees from different roles including Software Engineers, Product Managers, and UX designers came to me and asked for help. I also realized that, after I showed them how to use the current tools, eventually they would forget and came back to me again.

Solutions

  • One click to rule them all With ASCC, it is only take one click away to get all the seller feedback and page ratings. We were aiming to make it simple to get seller feedback and page ratings. Therefore, internal stakeholders can view page ratings, dive into specific feedback and comments from a seller, and view feedback trend throughout the year. With the insights from ASCC, they can plan for improvements and new features or bug fixes.

Outcomes and achievements

Our team received great feedback and got recognition from upper management and many employees from different organizations. This tool has set a new standard for internal tool and visualization tool teams. It has been actively using by many employees from different organizations.

Final implementation

The dashboard is coded using Bootstrap framework, Tableau JavaScript API, and C3.js chart library.

Due to the confidential nature of this project, examples will be provided upon request.

Next steps

In the next version, we plan to include other Seller metrics such as pages traffic and visitors.
